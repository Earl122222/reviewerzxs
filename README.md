## Technology Stack

- **Frontend**: React 18
- **Styling**: Tailwind CSS
- **Icons**: Font Awesome
- **Deployment**: Vercel

## Local Development

```bash
# Clone the repository
git clone <repository-url>
cd pnle-reviewer

# Start local server
python -m http.server 8000
# or
npx serve . -p 8000
```

Open http://localhost:8000 in your browser.

## Deployment

This app is deployed on Vercel and available at: https://pnle-reviewer.vercel.app
